### 文章默认头图

设置位置**后台->外观->设置外观**->**没有设置文章头图的就用这里的图片啦**

ps.如果单单填写张图片的地址可能略显枯燥。可以在此处填写你的随机图片api以达到更加美观的效果。

> 小技巧，当使用随机图片api时可能会出现重复获取同一张图片的情况，你可以使用`{random}`字段来避免这一问题
>
> 1.当你的随机图片api地址类似于`https://img.xx.xx/acg.php`，即api本身不含`?`时，只需要在地址末尾添加`?random={random}`即可。例如`https://img.xx.xx/acg.php?random={random}`
>
> 2.当你的随机图片api地址类似于`https://img.xx.xx/pic.php?cat=acg`，即api本身含有`?`时，则需要在地址末尾添加`&random={random}`。例如`https://img.xx.xx/pic.php?cat=acg&random={random}`

### 自定义小组件

设置位置**后台->外观->设置外观**->**侧边栏小组件配置**

> 注意，此设置会覆盖掉前面的“侧边栏小相片”、“侧边栏小视频”和“侧边栏图片描述”选项。

目前支持七种小组件

1. 图片展示
2. 视频展示
3. 链接组
3. 一言
3. 点赞
3. 最近评论
3. 随机文章

图片和视频有两个大小：`normal` 和 `large`

为了方便大家配置，做了一个小工具 [G主题小组件配置](https://g-editor.imouto.tech/)

![工具](https:\img.uaxk.com\pub\15\TWrcljAsXa.png)

你可以在左侧选择你想要的小组件并配置添加，添加后会在中间预览区中显示，最后复制右侧的json文件到后台保存就可以啦。

如果你是动手党，可以看看下面。这个是小组件的配置格式。

```json
[
    {...},//小部件1的配置
    {...},//小部件2的配置
    {...},//小部件3的配置
]
```

三种小组件的配置方式不相同，下面分别讲解

```JSON
//视频展示小组件
{
    "type": "video",
    "size": "large",
    "url": "https://cdn.exia.xyz/video/blog/nian.webm",
    "desc": "awsl!"
},

//图片展示小组件
{
    "type": "photo",
    "size": "normal",
    "url": "https://sdn.geekzu.org/avatar/87e0f8d2f0f70987061cec6376cb7f97?s=200&r=G&d=",
    "desc": "好热啊"
},

//链接组小组件
{
    "type": "cate",
    "size": "normal",
    "content": [ //content中填写需要展示的链接数组，可以是分类的链接，或者别的网页的链接。
        {
            "name": "闲扯",
            "url": "http://test-blog.test/category/nothing/"
        },
        {
            "name": "代码",
            "url": "http://test-blog.test/category/CODE/"
        },
        {
            "name": "ACG",
            "url": "http://test-blog.test/category/ACG/"
        }
        //数量必须为3个
    ]
},

//一言小组件
{
    "type": "hitokoto",
    "cate": "d,b" //语句的分类
},

//点赞小组件
{
    "type": "like",
    "size": "large"
},

//最近评论
{
    "type": "comments",
    "len": "3" //个数限制，1~10
},

//随机文章
{
    "type": "randomPost",
    "len": "4" //个数限制，1~你有的文章个数（建议别太多啦）
},
```

## 高级设置

这里是G主题的高级设置介绍。

高级设置填写规范：

```
key=value
```

样例：

![](https://cdn.exia.xyz//img/G_Doc/v3/65.png)

### 自定义ICP备案文字

```
customICP=xxx
```
xxx替换为你想要的文字
 > 只有当没有在主题设置中填写备案号时生效

### 自定义动画淡入时长
```
customAnimationInDuration=1s
```
1s替换为你想要的时长，必须加上单位s

### 自定义动画淡出时长
```
customAnimationOutDuration=0.2s
```
0.2s替换为你想要的时长，必须加上单位s

### 自定义赞助按钮文字
```
customSponsorText=喝杯茶
```
“喝杯茶”改为你想要的文字

### 自定义头部背景偏移
```
customHeaderOffsetX=-10px
customHeaderOffsetY=-10px
```
-10px改为你想要的数值，单位可以是任何CSS适用单位。规则：上加下减，左加右减
