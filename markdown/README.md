## 开始教程



### 标题<h>

```markdown
# 一级标题
## 二级标题
### 三级标题
...如此类推
```



### 列表<ul>,<ol>

**1.无序列表**

```markdown
 - 项目1
 - 项目2
 - 项目2
 ...
```

也可以使用

```markdown
 * 项目1
 * 项目2
 * 项目3
 ...
```

效果：

- 项目1
- 项目2
- 项目3

**2.有序列表**

```markdown
1. 项目1
2. 项目2
3. 项目3
...
```

效果：

1. 项目1
2. 项目2
3. 项目3



### 分割线<hr>

```markdown
***

---
```

效果：

***

---



### 引用<blockquote>

```markdown
 > 这是引用的话...
```

效果：

> 这是引用的话



### 标记

1.加粗<strong>

```markdown
**加粗字体哟**
```

效果：**变胖了！**

2.斜体<em>

```markdown
*斜体哟*
```

效果：*我怎么歪了*

3.删除线<del>

```markdown
~~阿伟死了~~
```

效果：~~阿伟死了~~

